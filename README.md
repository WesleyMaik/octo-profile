### Requirements

- node >= v16.14.1
- npm or yarn (optional)

### Built with

- ViteJS (React)
- ChakraUI
- React Query
- Chart.JS

### Getting Started

1. Clone the project

```
git clone https://github.com/WesleyMaik/octo-profile
```

2. Install dependencies
